Note: Apache Ant should be installed before the program can be run

Instructions to compile:

  1. Copy the jar file locally into your machine

  2. Unjar it by executing the following command : 
         $> jar xvf <jarfilename>
  3. Execute the following command to list all the targets available in the project :
         $> ant -projecthelp
  4. Execute the following command to compile the code :
         $> ant compile
  5. Execute the following command to generate javadoc :
         $> ant javadoc

Note:It is optional to run the commands 4 to compile. Running the below commands to execute the program will automatically do the compilation, packaging of JARs and deployment.

Instructions to start the compute space, computer and clients locally:

  1. Execute the following command to start the Compute Space :
         $> ant runLocalSpace
  2. Execute the following command to start the Computer :
         $> ant runLocalComputer
  3. Open a new shell and execute the following command to start a client to display a solution to the Travelling Salesman Problem on screen :
         $> ant runLocalTspClient -Dusername=<username>

Instructions to start the compute space, computer and clients on remote machines:

  1. Execute the following command to start the Compute Space in the remote host snoopy.cs.ucsb.edu :
         $> ant runRemoteSpace -Dusername=<username>
  2. Execute the following command to start one Computer at a remote host 
         $> ant run1RemoteComputers -Dusername=<username>
  3. Execute the following command to start four Computers at remote hosts 
         $> ant run4RemoteComputers -Dusername=<username>
  4. Execute the following command to start sixteen Computers at remote hosts 
         $> ant run16RemoteComputers -Dusername=<username>
  5. Open a new shell and execute the following command to start a client to display a solution to the Travelling Salesman Problem on screen by accessing a remote Compute Space :
         $> ant runTspClient -Dusername=<username>
